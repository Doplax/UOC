Descripción Star
En una Base de Datos relacional es importante establecer mecanismos de control para resolver problemas de concurrencia de usuarios y garantizar la seguridad de los datos. 

Para resolver el problema de la seguridad, trabajaremos con sentencias DCL para crear usuarios y asignarles privilegios y para gestionar la concurrencia aprenderemos a usar transacciones, las cuales profundizaremos en el Proyecto de Base de Datos.

Objetivos Event personal
Aprender las sentencias que permitan crear usuarios y asignarles privilegios.
Conocer el funcionamiento de las transacciones y poder anular parcial o totalmente los cambios producidos por una transacción.
