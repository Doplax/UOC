Descripción 
Como bien sabes en los siguientes productos se espera que seas capaz de editar una aplicación de consola en Python usando como entorno de desarrollo Visual Studio o VSCode. Pues bien, en este producto se trata de dejarlo todo preparado para que puedas trabajar con garantías de éxito en los productos siguientes.

Deberás realizar un video en formato MP4 de una duración máxima de 10 minutos en el que se muestre cómo crear un proyecto con Visual Studio Code para crear aplicaciones en modo consola escritas en lenguaje Python, se muestre también cómo depurarlas y ejecutarlas.

Objetivos 
Instalar un entorno de desarrollo y adquirir un dominio mínimo de las herramientas que éste nos ofrece..
