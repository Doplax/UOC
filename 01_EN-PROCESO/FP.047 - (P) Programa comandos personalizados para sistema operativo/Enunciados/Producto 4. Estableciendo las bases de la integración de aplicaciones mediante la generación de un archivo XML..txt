Descripción Star
Desarrollar una pequeña aplicación que sea capaz de extraer cierta información de diversas máquinas de la red, generar con ella un documento XML.

Objetivos Event personal
El objetivo principal del producto es:

Desarrollar una aplicación programada en Python que sea capaz de generar un archivo formato XML que contendrá diversas configuraciones de la máquina local.
Pasos a seguir  Pencil
Los pasos a seguir para llevar a cabo el producto son:

Leer detenidamente estas instrucciones e identificar los requerimientos de la actividad.
Revisar detenidamente la rúbrica de evaluación.
Consultar los recursos necesarios facilitados en el aula.
Programar una pequeña aplicación de consola escrita en Python, la funcionalidad de la cual será generar un fichero XML a partir de los siguientes datos del adaptador de red indicado por el usuario de la aplicación: la IP, máscara, puerta de enlace, el servidor DNS primario, la velocidad media de respuesta de este servidor DNS y del número de saltos a los que este servidor DNS se encuentra detallando también la IP de cada salto.  Mostrar con un submenú los adaptadores posibles donde el usuario escogerá, el que pondremos en el fichero XML
Hay que tener en cuenta que el XML generado tendrá que ser bien formado, es decir, es decir no ha de tener errores sintácticos.
Para su correcta realización la aplicación programada en Python debe estar:
Correctamente modularizada.
Su código ha de estar bien comentado.
Preguntar inicialmente el nombre del adaptador de red a analizar.
Generar el archivo XML con la información detallada anteriormente.
Preparar un documento PDF con la resolución de los ejercicios.
Preparar, para la entrega, un documento comprimido que incluya el documento PDF, los ficheros del programa en Python.
Se requiere Event personal
Los requisitos indispensables para realizar el producto son:

Haber realizado los productos 1, 2 y 3.

Conocer la tecnología XML.
