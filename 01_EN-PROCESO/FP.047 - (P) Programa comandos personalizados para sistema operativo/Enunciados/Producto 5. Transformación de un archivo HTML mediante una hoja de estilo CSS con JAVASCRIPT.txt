Descripción Star
A partir del documento XML generado en el producto 4, crear una web usando HTML, CSS i JavaScript que muestre la configuración actual de vuestro dispositivo de red y un botón que al pulsarlo muestre el HTML formateado correctamente con la hoja de estilo  XSLT en un formato atractivo para el usuario humano.

Objetivos Event personal
El objetivo principal del producto es:

Crear una web HTML que muestre configuración del dispositivo del archivo XML correctamente formateado, a partir de un botón o similar programado con JavaScript que al pulsarlo nos parezca la información correspondiente al archivo XML del producto anterior en la página web
Crear una hoja de estilo CSS para la transformación y presentación de un archivo formato HTML.
Pasos a seguir  Pencil
Los pasos a seguir para llevar a cabo el producto son:

Leer detenidamente estas instrucciones e identificar los requerimientos de la actividad.
Revisar detenidamente la rúbrica de evaluación.
Consultar los recursos necesarios facilitados en el aula.
Crear una hoja de estilo CSS para mostrar la información almacenada en el documento XML generado en el producto 4, de forma que maquete usando la estructura de una tabla la información contenida en el XML de una forma atractiva para un usuario que la requiera generando un archivo en formato HTML.
Crear una página web que muestre la configuración del dispositivo i un botón (programado con JavaScript) que permita cargar el XML creado en el producto 4 con la configuración CSS correspondiente.
Preparar un documento PDF con la resolución de los ejercicios.
Preparar, para la entrega, un documento comprimido que incluya el documento PDF, el fichero XML, archivo HTML y la hoja de estilo CSS.
Se requiere Event personal
Los requisitos indispensables para realizar el producto son:

Haber realizado los productos 1, 2, 3 y 4.
Conocer la tecnología XML y Javascript.
Conocer y construir lenguajes de transformación CSS.
Conocer básicamente la gramática HTML y Javascript.
Realizar la entrega de la transformación HTML en formato CSS.
